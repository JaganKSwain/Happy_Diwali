## Getting Started

Simply open `Happy_Diwali.html` in your web browser to view the celebration page. No installation or dependencies required!

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Running the Project

**Option 1: Direct File Opening**
1. Clone or download this repository
2. Open `Happy_Diwali.html` in your browser
3. Enjoy the festive display!

**Option 2: Using Python HTTP Server**
1. Navigate to the project directory in your terminal
2. Run the following command:
   ```bash
   python3 -m http.server
   ```
3. Open your browser and go to `http://localhost:8000`
4. Click on `Happy_Diwali.html` to view the celebration

## Features Explained

### Visual Elements
- **Diya Icon**: Traditional oil lamp emoji representing Diwali
- **Main Message**: Centered greeting with golden text
- **Wishes Text**: Celebratory message in uppercase letters
- **Festive Colors**: Golden (#d4af37), silver (#c0c0c0), and dark background

### Animations
- **Fade-in Effects**: Staged animation of text elements with varying delays
- **Fireworks**: Multi-colored particles that explode and fade away
- **Sky Shots**: Rising particles creating a celebratory atmosphere

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Animations, gradients, and responsive styling
- **Vanilla JavaScript**: Canvas-based graphics and animation logic

## Customization

You can easily customize:
- Animation timings by modifying the CSS animation values
- Colors by changing the hex color codes
- Messages by editing the text content in the HTML
- Fireworks intensity by adjusting JavaScript variables

## About Diwali

Diwali, also known as Deepavali or Lakshmi Puja, is a significant festival celebrated in India and across the world. It symbolizes the victory of light over darkness and good over evil. The festival is marked by the lighting of diyas, fireworks, decorations, and the sharing of sweets.
